# 主張台帳

成果物で使う主張・数字はここに登録してから使う。成果物側での発明は行わない。

- 成熟度 `validated`：出典確認済み。外部提出物の本文に使える
- 成熟度 `pending`：出典待ち。内部ドラフトのみ
- 成熟度 `desired`：あれば説得力が上がる数字。取りに行く調査リストとして扱う
