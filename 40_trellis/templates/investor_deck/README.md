# 投資家資料テンプレート

ここに投資家資料のテンプレート群を置く。

## 想定するテンプレート

- `slide_map.template.yaml`：スライド構成の雛形
- `slide_spec.template.md`：1 スライド分の仕様書雛形
- `speaker_notes.template.md`：speaker note の雛形
- `deck_brief.template.md`：その回のブリーフ雛形
- `context_pack.template.md`：その回の文脈凝縮の雛形

> MVP 段階では雛形は未投入。最初の投資家資料 v001 を作る過程で `50_greenhouse/investor_deck/` 内に直接書き、安定したらここに昇格させる。
